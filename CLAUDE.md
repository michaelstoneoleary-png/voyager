# Voyager

This is the **Voyager** travel planning app. The folder is named `voyager` at `/home/user/voyager`.

> There is an unrelated project at `/home/user/conductor`. All development work for Voyager must be done in this repo only — never in conductor.

## Project structure

- `client/` — React + Vite frontend
- `server/` — Express backend
- `mobile/` — Expo React Native app
- `shared/` — Shared types and schema (Drizzle)

## Dev commands

- `npm run dev` — start frontend + backend together
- `cd mobile && npx expo start` — start mobile app
